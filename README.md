## What this is

Generad is a sensibilisation project demonstrating how trivially easy it is to generate convincing fake UGC (User-Generated Content) video reviews using off-the-shelf AI tools. You provide a product name, a description, and an image. The pipeline produces a photorealistic reviewer, a natural-sounding script, and a smartphone-aesthetic video review — indistinguishable from content posted daily on TikTok, Instagram, and YouTube.

No production skills required. No actors. No cameras. No budget.

This was built as an awareness tool after being approached to build this commercially around the end of 2025. I turned that down. I'm publishing this instead.

---

## Why it matters

Consumer trust in UGC reviews is high precisely because they appear unscripted and personal. In fact, their scrappy nature makes them feel authentic, but is also what allows AI generations to find their place. This project shows that appearance can be fully synthetic:

- The reviewer doesn't exist — AI-generated persona and photorealistic image
- The script was never written by a human — GPT-generated with deliberate "authentic" imperfections
- The video was never filmed — generated by a video model conditioned on a fake first frame

The tools used here are publicly available, inexpensive, and require no ML expertise to operate. Plus, implementing this with agents like Claude Code or OpenAI's Codex is blazingly fast. The barrier to deploying fake reviews at scale is effectively zero.

---

## The pipeline

Five steps, fully automated:

1. **Persona generation** — GPT-4o-mini generates a detailed, credible reviewer profile tailored to the product
2. **Creator image** — Fal.ai generates a photorealistic selfie of the fake reviewer
3. **First frame** — Fal.ai composites the creator holding the product into a single opening shot
4. **Video script** — GPT-4o-mini writes a timestamped 14-second UGC script with natural speech patterns, filler words, and self-corrections
5. **Video generation** — Google Veo 3.1 generates an 8-second first half from the first frame and script, then extends it into a full video

Total generation time: ~2-3 minutes, dominated by video generation.

Total generation cost: ~$1.

## Setup

**Requirements:** Python 3.8+, API keys for OpenAI, Google Gemini, and Fal.ai.

```bash
git clone <repository-url>
cd Generad
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
streamlit run streamlit_app.py
```

Copy `.env.example` to `.env` and fill in your keys, or enter them directly in the app sidebar:

```
OPENAI_API_KEY=...
GEMINI_API_KEY=...
FAL_KEY=...
```

## Tech stack

- [OpenAI GPT-4o-mini](https://platform.openai.com/) — persona and script generation

This project exists to demonstrate a capability, not to enable it. Using AI to generate fake reviews may violate consumer protection law, platform terms of service, and constitute false advertising in your jurisdiction.
